# 東方夜魔傳（東方闇魔伝）

This is a TouHou like game, based on Vue.js without any animation library.

![EDU2021](src/assets/logo.svg)

Respect to `東方紅魔鄉 ～ the Embodiment of Scarlet Devil`.


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
